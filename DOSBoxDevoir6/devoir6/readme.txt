Ce kit de développement fonctions sour DOS (Windows -> invite de commande).

FCEUXD.EXE : émulateur NES.  Offre des options de débuggage.

NESASM.EXE : compilateur NES.  Utilisation: nesasm fichier.asm

TLP.EXE : Tile Layer Pro, logiciel d'édition des tuiles.  Ne devrait pas vous servir, mais peut
vous aider à retrouver des morceaux d'images parmi les tuiles de votre ROM.
Les tuiles se trouvent dans le fichier "smbvrom.nes".
